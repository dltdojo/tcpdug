# PKC

為了讓群組分享激勵較為簡單例如贊助一杯咖啡，這時需要有個容易說明的換算單位，故採用普朗克咖啡 ( Planck Coffee )為換算單位，利用普朗克常數來計算波卡生態圈內代幣近似一杯普通咖啡的換算單位，簡稱普咖也就是PKC。

一杯普咖 = 1 PKC = 6,626,070,150 Planck = 0.662607015 DOT (可以想像約一杯咖啡價)
